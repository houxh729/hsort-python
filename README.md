# hsort

HSORT is a high-performance sorting and merging engine for large-scale data files. It supports fixed-length, variable-length, and CSV formats, stable multi-key sorting, and is designed to efficiently handle multi-gigabyte datasets with minimal memory fragmentation.

> ⚠️ This project is under active development.  
> APIs and behavior may change without notice.

## Features

- High-performance sorting and merging for large-scale data files
- Supports fixed-length, variable-length, and CSV formats
- Stable multi-key sorting
- Designed for efficient processing of multi-gigabyte datasets
- Optimized memory usage with minimal fragmentation
